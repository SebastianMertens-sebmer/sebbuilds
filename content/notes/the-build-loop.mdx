My default loop is simple: find the smallest honest version of an idea, build it, publish it, and use what happens next as the brief for the next version.

The loop works because it keeps taste connected to evidence. It also makes progress visible.

## The loop

1. Write down the problem in plain language.
2. Build the smallest useful surface.
3. Put it where someone can actually use it.
4. Study the friction and improve the next slice.
